# WpfChartDemo

ゲーム利用者分析のためのWPFアプリケーション

## 機能

このアプリケーションは、ゲームの利用者分析を行うための3つのチャートを提供します：

1. **年齢別ゲーム利用者数** - 男性・女性別の年代別利用者数を棒グラフで表示
2. **月別売り上げ** - 年間の月別売り上げ推移を折れ線グラフで表示  
3. **国別売り上げ** - 世界各国の売り上げを横棒グラフで表示

## 使用技術

- WPF (.NET 8.0)
- LiveChartsCore.SkiaSharpView.WPF v2.0.0-rc5.4

## データ

現在はサンプルデータを使用しています。将来的にはローカルのExcelファイルからデータを読み込む機能を追加予定です。

## 実行方法

```bash
dotnet build
dotnet run
```

**注意**: このアプリケーションはWindows環境でのみ実行可能です（WPF依存のため）。
